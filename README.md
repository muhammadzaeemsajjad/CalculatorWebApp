Retro-Vibe Web Calculator
This vibrant Retro-Vibe Calculator features a bold pink-on-cyan aesthetic with a minimalist UI. It provides a 4x5 tactile grid supporting arithmetic, exponents, and parentheses. Designed for clarity and style, it combines a high-contrast display with responsive logic, making it a functional piece of digital eye candy for any web portfolio.

Features
Arithmetic Operations: Supports basic math using JavaScript's eval() logic.

Power Functionality: Custom handling for the ^ operator to perform exponentiation.

Instant Clear: Quick-reset button ("C") for efficient workflow.

Interactive UI: Smooth hover effects with a distinct red shadow highlight.

Tech Stack
HTML5: Semantic structure for the calculator body and display.

CSS3: Custom styles using vh/vw units for a responsive, screen-centered layout.

JavaScript (ES6): Event-driven logic using querySelectorAll and forEach to manage user input.

File Structure
index.html: The core structure of the calculator.

style.css: The styling logic (synthwave/retro theme).

script.js: The functional logic for mathematical calculations.

How to Run
Clone this repository to your local machine.

Open index.html in any modern web browser.

Start calculating!
